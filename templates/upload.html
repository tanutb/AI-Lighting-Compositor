How to use:

  1. Upload an Image: Drag a photo or render into the box above.
  2. Set API Key: In the workspace, click "API Settings" (top menu) and enter your Gemini API Key.
  3. Add Lighting: Click "+ Add Layer", type a prompt (e.g., "blue rim light"), and click Generate.
  4. Adjust: Use the opacity slider to blend the lighting effect.
